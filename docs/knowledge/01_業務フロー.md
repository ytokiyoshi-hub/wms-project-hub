# 01 業務フロー

**カテゴリ：倉庫業務の工程・作業フロー**  
**最終更新：2026-05-07**  
**ステータス：骨格ドラフト**

---

## このドキュメントの目的

WMSが扱う倉庫業務の「工程の流れ」を全員が理解できるよう整理する。  
3PL（複数荷主受託型）特有の考え方も含む。

---

## 全体フロー概観

```
【荷主】 → 出荷指示 → 【WMS】 → 作業指示 → 【倉庫現場】 → 実績送信 → 【荷主】
                      ↑
                   在庫管理
                      ↑
                  【入庫処理】
```

WMSの中心は「在庫の入口から出口まで」の一貫管理。

---

## 1. 入庫工程

### 1-1 入荷予定の受付

- 荷主から入荷予定データを受信（EDI・手入力）
- 入荷予定がない場合は「飛込み入荷」として対応

### 1-2 荷受け（受入）

- トラックからの荷降ろし
- 外観確認・数量確認
- 検品（破損・品番・ロット番号）

### 1-3 格納

- 入荷品に格納先ロケーションを割り当て
- ハンディターミナル（HT）でロケーションをスキャン
- 格納先の種別：パレット棚 / フリーロケ / 仮置き

```
ロケーションコード例：3AP-02-134
→ 3階 / Aフロア / パレット棚 / 02通路 / 1列3段4位置
```

### 1-4 在庫計上

- WMS上で在庫数が確定
- FIFO（先入れ先出し）原則に基づき、入荷日・ロットを記録

---

## 2. 在庫管理工程

### 2-1 ロケーション管理

- 在庫は「ロケーション単位」で管理（どこに何がいくつあるか）
- ロケーション種別：パレット(P)・棚バラ(T)・流動棚(F)・仮置き(K)・不適合品(R)・ハイラック(H)

### 2-2 在庫ステータス管理

| ステータス | 意味 |
|-----------|------|
| 通常 | ピッキング引当可能 |
| ロット出荷止め | 問題のあるロット。引当不可 |
| 引当除外 | 特定ロケの在庫を引当対象から外す |

### 2-3 補充（リプレニッシュメント）

- ピッキングロケーションの在庫が閾値を下回ったら自動補充指示
- 補充元：バックストック（パレットロケ）→ 補充先：ピッキングロケ（棚・流動棚）
- 補充タイミング：閾値ベース or 出荷実績ベースの自動計算

### 2-4 棚卸し

- 定期棚卸し（月次・年次）
- 不定期棚卸し（差異発生時）
- HT で実棚数を入力 → WMS残高と照合

---

## 3. 出荷（出庫）工程

### 3-1 出荷指示受付

- 荷主からの出荷指示を受信（EDI・手入力・ポータル入力）
- **注意**：WMSにおける「受注」は「出荷指示」と呼ぶ（受注という概念は使わない）

### 3-2 引当

- 在庫をピッキング対象として確保
- FIFO強制：入荷日の古い在庫から優先引当
- 引当方式（受注ごとに選択）：
  - **エリア並行方式**：複数作業者が同時進行
  - **ゾーンリレー方式**：エリアを順番にリレー

### 3-3 ウェーブ分割・作業指示

- 複数の出荷指示を束ねて「ウェーブ」として一括処理
- ウェーブをエリア別・温度帯別に分割
- HTに作業指示を送信

### 3-4 ピッキング

- HTの指示に従い、ロケーションから商品を取り出す
- FIFO順にピッキング（システムが順番を制御）
- バラ（ピース）とケースの並行作業対応

### 3-5 梱包

- ピッキングした商品を梱包材に詰める
- 梱包材は商品・荷主別のルールに基づき自動選定
- 梱包確定により個口数が確定 → 荷札が発行される

### 3-6 仮置き・集荷待ち

- 梱包済みカゴ・パレットを仮置きエリアへ
- カゴIDで管理（どの便に積むか紐付け）
- 集荷時刻アラートで積込作業を開始

### 3-7 積込（引渡し）

- 配送業者のトラックに積み込み
- 積込明細書を発行・確認
- 引渡し記録（荷主への実績報告用）

### 3-8 出荷実績送信

- 配送伝票番号・出荷結果を荷主システムへ送信
- 業界標準フォーマット（EDI等）対応
- エラー発生時はアラート通知

---

## 4. 3PL特有の業務フロー

### 4-1 荷主切替

- 複数荷主の在庫が同一倉庫に混在
- 担当者は作業中に荷主を切り替えながら対応
- データは荷主別に完全分離

### 4-2 荷主ポータル

- 荷主自身がWebポータルで出荷指示・在庫照会
- 荷主ごとにカスタマイズ可能な表示

### 4-3 3PL特有の請求業務

- 荷主ごとの保管料・荷役料の計算
- 保管料計算方式：3期制 / 2期制 / 坪貸し
- 請求書発行・明細提示

---

## 5. HT（ハンディターミナル）が使われる工程

| 工程 | HT使用場面 |
|------|-----------|
| 入庫 | 格納ロケーションのスキャン |
| 在庫 | 棚卸し実棚入力 |
| 出荷 | ピッキング指示・スキャン検品 |
| 出荷 | 積込確認 |
| 出荷 | 引渡し確認 |

対応機種：キーエンス BT-A2000（4インチ）/ BT-A1000（3インチ）
