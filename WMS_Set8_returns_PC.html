W-1003返品良品 在庫戻し承認 Q10-4
概要：返品良品（inventory.status = 'returned_good'）を通常在庫（'available'）に戻す承認画面。荷主設定 owners.return_strategy.auto_restock = false の場合のみ承認操作が必要。
auto_restock = true の荷主はHT検品完了時に自動で available に移行するため、この画面には表示されない。倉庫スタッフも承認可能（荷主ロール不要）。

W-1004廃棄最終承認（荷主ロールのみ） Q10-5
概要：倉庫スタッフがHTで行った廃棄申請を荷主が最終承認する画面。法的根拠：寄託物の所有権は荷主にあるため、廃棄の最終確定は荷主の承認が必須（Q10-5：選択肢C確定）。
権限：荷主ロール（shipper）のみ操作可能。倉庫ロールはPC画面でも申請閲覧のみ（HT版と同じ申請権限）。PC版のほうがHTより承認権限が強い（荷主ログイン時のみ最終承認ボタンが有効）。
https://wms.example.com/returns/disposal-approval
