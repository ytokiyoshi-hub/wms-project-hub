# WMS Project Hub

3PL（複数荷主受託型）対応の倉庫管理システム（WMS）の仕様策定・開発進捗管理ハブです。

🌐 **公開サイト**: [https://ytokiyoshi-hub.github.io/wms-project-hub/](https://ytokiyoshi-hub.github.io/wms-project-hub/)

---

## 概要

このリポジトリは、WMS(倉庫管理システム)の仕様書・画面モック・進捗状況を一元管理するためのプロジェクトポータルです。

- **対象システム**: WMS(倉庫管理システム)3PL対応版
- **ベースシナリオ**: 株式会社マルキ食品(食品卸 BtoB+EC、架空企業)
- **HT機種**: キーエンス BT-A2000 / BT-A1000
- **現在のフェーズ**: 仕様策定の最終段階(v3.1)

## 含まれるコンテンツ

| カテゴリ | 内容 |
|---|---|
| ダッシュボード | プロジェクト全体の進捗状況 |
| 画面モック | 65画面(PC・HT・スマホ・荷主ポータル) |
| ドキュメント | 仕様書・API仕様書・ER図・操作マニュアル等 8種 |
| 機能リスト | 確定済み追加機能 98件をカテゴリ別に管理 |
| 確定事項 | 設計確定事項(E-1〜E-6)と業務ルール |
| 未決事項 | 今後のレビューで詰める項目 |

## 使い方

トップページの `index.html` を起点に、サイドバーから各セク
