# WMS Project Hub

3PL（複数荷主受託型）対応の倉庫管理システム（WMS）の仕様策定・開発進捗管理ハブです。

🌐 **公開サイト**: [https://YOUR-ORG.github.io/wms-project-hub/](https://YOUR-ORG.github.io/wms-project-hub/) ※公開後にURL更新

---

## 概要

このリポジトリは、WMS（倉庫管理システム）の仕様書・画面モック・進捗状況を一元管理するためのプロジェクトポータルです。

- **対象システム**: WMS（倉庫管理システム）3PL対応版
- **ベースシナリオ**: 株式会社マルキ食品（食品卸 BtoB+EC、架空企業）
- **HT機種**: キーエンス BT-A2000 / BT-A1000
- **現在のフェーズ**: 仕様策定の最終段階（v3.1）

## 含まれるコンテンツ

| カテゴリ | 内容 |
|---|---|
| ダッシュボード | プロジェクト全体の進捗状況 |
| 画面モック | 65画面（PC 53・HT 5・スマホ 3・荷主ポータル 9） |
| ドキュメント | 仕様書・API仕様書・ER図・操作マニュアル等 8種 |
| 機能リスト | 確定済み追加機能 98件をカテゴリ別に管理 |
| 確定事項 | 設計確定事項（E-1〜E-6）と業務ルール |
| 未決事項 | 今後のレビューで詰める項目 |

## 使い方

トップページの `index.html` を起点に、サイドバーから各セクションに移動できます。

- **ダッシュボード**: プロジェクト全体の俯瞰
- **画面モック**: クリックで該当画面を別タブで開く
- **ドキュメント**: 仕様書のダウンロード
- **追加機能リスト**: 検索可能な98機能の一覧

## ローカルで開く

GitHubから zip でダウンロードするか、git clone した後、`index.html` をブラウザで開いてください。

```bash
git clone https://github.com/YOUR-ORG/wms-project-hub.git
cd wms-project-hub
open index.html  # macOS
# または index.html をダブルクリック
```
