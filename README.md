# Design References

実装フェーズで参考にするデザインのストック置き場です。

仕様策定中の画面モック（Set1〜Set5、GroupA〜C等）は機能・画面構成の確認用として簡素なデザインで作っていますが、実装時にはこのフォルダの参考デザインのような今風で洗練された見た目で作りたいと考えています。

## 含まれるファイル

### wms_design_reference.html

WMS Web画面の3画面（ダッシュボード、在庫一覧、引当・ウェーブ）のデザイン参考。

**特徴**:
- 明るく清潔感のあるデザイン
- 検索フォームに余白を取った見やすいレイアウト
- アクションボタンの色分け（CSV出力・印刷など）
- データテーブルの行間が広く、視線が流れやすい
- 紺色サイドバーと白基調メインエリアのコントラスト

このファイルをブラウザで開くと、デザインの雰囲気が確認できます。

## 実装フェーズでの使い方

Claude Codeで実装する際、以下のように指示する想定：

> このdesign-references/wms_design_reference.htmlのような雰囲気で、React + Tailwind CSS + shadcn/uiで実装してください。
> 
> 機能や画面構成は、別途 WMS_Set1〜Set5 等のモック画面を参照してください。

## 推奨技術スタック（参考）

- **フレームワーク**: React + TypeScript
- **CSSフレームワーク**: Tailwind CSS
- **UIコンポーネント**: shadcn/ui（このデザインに最も近い雰囲気）
- **アイコン**: Lucide React
- **フォント**: Noto Sans JP + Inter

## 今後の追加方針

実装が進む中で「こんなデザインにしたい」という参考画像を見つけたら、このフォルダに追加していきます。

例：
- ダッシュボードの参考デザイン
- グラフ・チャートの参考
- フォーム入力の参考
- モバイル画面の参考

ファイル名のルール（推奨）：
- `[用途]_[簡単な説明].html` または `.png`
- 例：`dashboard_modern.html`、`form_design.png`
